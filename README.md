# myDiploma2

Mobile application on React Native for conducting sightseeing quests

## Get started

1. Install [NodeJS](https://nodejs.org/en/download)

2. Clone this project

   ```bash
   git clone https://github.com/SalavatIskhakov/myDiploma2.git
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the app

   ```bash
   npm start
   ```

5. Install [Expo Go](https://expo.dev/go) on your smartphone and scan the QR code from the terminal
